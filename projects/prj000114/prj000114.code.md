## Core Classes

### JWTEncoder
**File**: `jwt_manager.py`, lines 29-120

**Responsibility**: Encodes and decodes JWT tokens with HMAC-SHA256
**Key Methods**:
1. `encode(claims)`: Create JWT token from claims dict
2. `decode(token)`: Decode and verify token
3. `_base64_urlsafe_encode(data)`: Encode without padding
4. `_base64_urlsafe_decode(data)`: Decode without padding
5. `_sign(message)`: Create HMAC-SHA256 signature
6. `_verify_signature()`: Compare signatures safely

**Token Structure**:
```
header.payload.signature
```

Where each part is base64url-encoded JSON.

**Algorithm**: HMAC-SHA256 (HS256)
**Error Handling**: Returns (success, data, error_message)

### TokenValidator
**File**: `jwt_manager.py`, lines 123-160

**Responsibility**: Validates token claims and expiration
**Key Methods**:
1. `validate(token)`: Check if token is valid
2. `_check_expiration(claims)`: Verify not expired
3. `extract_claims(token)`: Get claims from token

**Validation Checks**:
- Required claims present
- Expiration time in future
- Token structure valid

### RefreshTokenManager
**File**: `jwt_manager.py`, lines 163-238

**Responsibility**: Manages access and refresh token pairs
**Key Methods**:
1. `generate_token_pair(user_id)`: Create access + refresh tokens
2. `validate_refresh_token(token)`: Verify refresh token
3. `rotate_tokens(refresh_token)`: Generate new pair from refresh
4. `get_access_time_remaining(token)`: Check access token TTL

**Token Timings**:
- Access Token: 15 minutes
- Refresh Token: 7 days
- Total lifetime: ~7 days via rotation

**Usage**:
```python
mgr = RefreshTokenManager(encoder)
access, refresh = mgr.generate_token_pair("user123")

# Later: rotate when access expires
new_access, new_refresh = mgr.rotate_tokens(refresh)
```

## Implementation Patterns

### Base64 URL-Safe Encoding
```python
def _base64_urlsafe_encode(self, data):
    # Remove padding, use - and _ instead of + and /
    return base64.urlsafe_b64encode(
        data.encode('utf-8')
    ).decode('utf-8').rstrip('=')
```

### Signature Verification
```python
# Use constant-time comparison to prevent timing attacks
if not hmac.compare_digest(signature_encoded, expected):
    return False, None, "Invalid signature"
```

### Expiration Checking
```python
if claims.get('exp', 0) <= int(datetime.utcnow().timestamp()):
    return False, "Token expired"
```

## Usage Example

```python
from jwt_manager import JWTEncoder, RefreshTokenManager
from datetime import datetime

# Create encoder
encoder = JWTEncoder("your-secret-key")

# Generate token pair
mgr = RefreshTokenManager(encoder)
access, refresh = mgr.generate_token_pair("user123")

# Validate access token
is_valid, claims, error = encoder.decode(access)
if is_valid:
    print(f"User: {claims['sub']}")

# Rotate when access expires
new_access, new_refresh = mgr.rotate_tokens(refresh)
```
