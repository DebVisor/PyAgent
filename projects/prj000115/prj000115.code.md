## Core Class

### GlobalState
**File**: `global_state.py`, lines 43-227

**Responsibility**: Thread-safe global state container with observer pattern
**Key Methods**:

1. `get(path, default)`: Retrieve state value
   - Supports dot notation: "user.name"
   - Thread-safe read with RLock
   - Returns deep copy to prevent external mutation

2. `set(path, value)`: Update state value
   - Creates StateChange event
   - Notifies all subscribers
   - Records change in history
   - Thread-safe with RLock

3. `subscribe(observer)`: Add change listener
   - Observer receives StateChange events
   - Callable[[StateChange], None]

4. `unsubscribe(observer)`: Remove listener

5. `create_snapshot()`: Save current state
   - Deep copy for immutability
   - Includes change count

6. `restore_snapshot(snapshot)`: Restore to previous state
   - Replaces current state
   - Maintains change history

7. `get_change_history()`: Retrieve all changes
   - Limited to max_history_size (1000)
   - Removes oldest when limit reached

8. `clear_history()`: Clear change history

## Implementation Patterns

### Thread-Safe Dictionary Update
```python
with self._lock:
    keys = path.split(".")
    current = self._state
    for key in keys[:-1]:
        if key not in current:
            current[key] = {}
        current = current[key]
    current[keys[-1]] = value
```

### Nested Path Navigation
```python
keys = path.split(".")
current = self._state
for key in keys:
    if isinstance(current, dict) and key in current:
        current = current[key]
    else:
        return default
return deepcopy(current)
```

### Observer Notification
```python
change = StateChange(
    timestamp=datetime.now().isoformat(),
    path=path,
    old_value=old_value,
    new_value=value
)
self._change_history.append(change)
for observer in self._observers:
    observer(change)
```

## Usage Example

```python
from global_state import GlobalState

# Create global state
state = GlobalState()

# Simple set/get
state.set("theme", "dark")
theme = state.get("theme")

# Nested properties
state.set("user.name", "Alice")
name = state.get("user.name")

# Observer pattern
def on_change(change):
    print(f"{change.path}: {change.old_value} → {change.new_value}")

state.subscribe(on_change)
state.set("count", 42)  # Prints: count: None → 42

# Snapshots
snapshot = state.create_snapshot()
state.set("temp", "value")
state.restore_snapshot(snapshot)
```
