### Core Classes

#### ComponentStatus (Enum)
Status tracking for components:
- IDLE: Waiting
- RUNNING: Executing
- ERROR: Error state
- STOPPED: Shutdown

#### ComponentConfig (Dataclass)
Configuration with:
- name: Component ID
- timeout: Execution timeout
- max_retries: Retry limit
- enabled: Enable flag
- metadata: Custom data

#### ComponentResult (Dataclass)
Result with:
- success: Boolean status
- data: Result payload
- error: Error message
- duration_ms: Duration
- timestamp: Time
- metadata: Result metadata

#### IComponent (ABC)

- initialize(config)
