## Data Structures

### BenchmarkResult (Dataclass)
**File**: `benchmark_runner.py`, lines 14-35

**Responsibility**: Represents a single benchmark execution result
**Fields**:
- `name: str` - Benchmark name
- `iterations: int` - Number of iterations
- `mean_ns: float` - Mean time in nanoseconds
- `std_dev_ns: float` - Standard deviation
- `min_ns: float` - Minimum time
- `max_ns: float` - Maximum time
- `timestamp: str` - ISO format timestamp

**Methods**:
1. `throughput(items_per_iter)`: Calculate items/second
   - Formula: (1e9 / mean_ns) * items_per_iter
   
2. `outlier_percentage()`: Estimate outlier percentage
   - Formula: (std_dev / mean) * 100

**Usage**:
```python
result = BenchmarkResult(
    name="vector_sort",
    iterations=1000,
    mean_ns=5000.0,
    std_dev_ns=500.0,
    min_ns=4500.0,
    max_ns=5500.0,
    timestamp=datetime.now().isoformat()
)
throughput = result.throughput(1)  # items/sec
outlier_pct = result.outlier_percentage()
```

## Core Classes

### BenchmarkRunner
**File**: `benchmark_runner.py`, lines 38-116

**Responsibility**: Executes Criterion benchmarks and parses results
**Key Methods**:
1. `__init__(project_dir)`: Initialize with Rust project directory
2. `run_benchmarks(benchmark_name)`: Execute benchmarks via cargo
3. `_parse_results(output)`: Parse Criterion output
4. `_convert_to_ns(value, unit)`: Convert time to nanoseconds
5. `get_results()`: Retrieve parsed results

**Execution Flow**:
1. Build cargo bench command
2. Execute with 300-second timeout
3. Capture stdout/stderr
4. Parse output line by line
5. Extract timing information
6. Create BenchmarkResult objects

**Time Unit Conversion**:
```
'ns' → 1.0
'us' / 'µs' → 1e3
'ms' → 1e6
's' → 1e9
```

**Error Handling**:
- FileNotFoundError: cargo not found
- subprocess.TimeoutExpired: benchmark timeout
- ValueError: invalid timing format
- All errors return (False, error_message)

### ResultsAnalyzer
**File**: `benchmark_runner.py`, lines 119-190

**Responsibility**: Analyzes benchmark results and detects regressions
**Key Methods**:
1. `__init__(results)`: Initialize with BenchmarkResult list
2. `compare_with_baseline(baseline_results, threshold_percent)`: Compare against baseline
3. `get_slowest_benchmarks(count)`: Get top N slowest benchmarks
4. `get_fastest_benchmarks(count)`: Get top N fastest benchmarks
5. `calculate_statistics()`: Compute overall statistics

**Regression Detection**:
- Calculates percent change: `(current - baseline) / baseline * 100`
- Marks as regression if `|percent_change| > threshold`
- Categories: "slowdown" (positive), "speedup" (negative), "OK"

**Comparison Output**:
```python
{
    "baseline_ns": 5000.0,
    "current_ns": 5250.0,
    "percent_change": 5.0,
    "is_regression": True,
    "regression_type": "slowdown"
}
```

**Statistics Calculation**:
- Count: number of benchmarks
- Total time: sum of all mean_ns
- Mean: average of all benchmarks
- Min/Max: extremes across all results

### ReportGenerator
**File**: `benchmark_runner.py`, lines 193-252

**Responsibility**: Generates benchmark reports in multiple formats
**Key Methods**:
1. `__init__(analyzer)`: Initialize with ResultsAnalyzer
2. `generate_json_report()`: Create JSON report
3. `generate_text_report()`: Create formatted text report

**JSON Report Structure**:
```json
{
  "generated_at": "ISO timestamp",
  "statistics": {
    "count": 5,
    "total_time_ns": 25000,
    "mean_ns": 5000,
    "min_ns": 4500,
    "max_ns": 5500
  },
  "benchmarks": [
    {
      "name": "...",
      "mean_ns": ...,
      "std_dev_ns": ...,
      "throughput": ...
    }
  ]
}
```

**Text Report Sections**:
1. Header with border
2. Overall Statistics (count, mean, min, max)
3. Slowest Benchmarks (top 5)
4. Regressions (if comparisons exist)
5. Formatted columns and alignment

## Implementation Patterns

### Timing Parsing Pattern
```python
# Extract from "name time: [123.45 us]"
if 'time:' in line and '[' in line:
    name = line.split('time:')[0].strip().split()[-1]
    timing = line.split('time:')[1].strip().split()[0]
    time_value, time_unit = timing.split()
    time_ns = self._convert_to_ns(float(time_value), time_unit)
```

### Regression Comparison Pattern
```python
percent_change = ((current - baseline) / baseline) * 100
is_regression = abs(percent_change) > threshold
regression_type = "slowdown" if percent_change > 0 else "speedup"
```

### Report Formatting Pattern
```python
lines = ["═" * 60, "BENCHMARK REPORT", "═" * 60, ""]
lines.append("OVERALL STATISTICS")
lines.append(f"  Mean: {stats['mean_ns']:.2f} ns")
```

## Usage Example

```python
from benchmark_runner import (
    BenchmarkRunner, ResultsAnalyzer, ReportGenerator
)

# Run benchmarks
runner = BenchmarkRunner("./rust_core")
success, msg = runner.run_benchmarks("my_bench")

if success:
    results = runner.get_results()
    
    # Analyze results
    analyzer = ResultsAnalyzer(results)
    slowest = analyzer.get_slowest_benchmarks(5)
    stats = analyzer.calculate_statistics()
    
    # Generate reports
    generator = ReportGenerator(analyzer)
    json_report = generator.generate_json_report()
    text_report = generator.generate_text_report()
    print(text_report)
```
