# Projects

This file is a stable landing page for the **project portfolio** inside this repository.

## Purpose
- Provide a consistent, agent-readable place to find the list of active projects
- Encourage agents to write or update `docs/project/prjXXX-*` folders when new work is proposed or started

## How to use
- New projects should create a `docs/project/prjNNN-name/` folder with:
  - `brainstorm.md` (intent + reasoning)
  - `plan.md` (execution plan)

## Where to find existing projects
- See the `docs/project/` directory for all current project folders.
